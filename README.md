# Aerial Detection

This project is aimed at detecting objects in aerial images using various machine learning techniques. 

## Deployment Section

The model is deployed using **Streamlit** for the frontend interface.
For interface, the model is converted to **ONNX FORMAT** and executed using **ONNX Runtime** instead of TensorFlow.  
This significantly reduces dependency size and improves performance, making the application more suitable for lightweight deployment environments like Streamlit Cloud.
